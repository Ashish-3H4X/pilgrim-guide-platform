# Pilgrim Guide Platform

Pilgrim Guide is a smart local travel guide platform designed for pilgrimage cities in India.

The goal of this platform is to help travelers easily find temples, free stay (dharamshala), food places, and important attractions when they arrive in a city.

## Target Cities

- Mathura
- Vrindavan
- Ayodhya
- Varanasi
- Haridwar
- Ujjain

## Core Features

- Temple information
- Temple timing and rules
- Nearby famous places
- Free stay (dharamshala)
- Free food / bhandara
- Budget hotels
- Suggested travel routes
- Map navigation
